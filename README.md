# Telegram Morning Bot

Этот бот отправляет пожелание доброго утра пользователю каждый день в заданное время.

## Использование

1. Установите зависимость `python-telegram-bot`:
   ```bash
   pip install python-telegram-bot
   ```
2. Замените `YOUR_TELEGRAM_BOT_TOKEN` в `morning_bot.py` на токен вашего бота.
3. Запустите скрипт:
   ```bash
   python morning_bot.py
   ```
4. Напишите вашему боту команду `/start`, чтобы активировать ежедневные сообщения.

По умолчанию сообщение отправляется в 9:00 утра (временная зона сервера).
